# Smart Parking Slot System

## Description
This project detects parking slot availability using an ultrasonic sensor and Arduino.

## Features
- Detects car presence
- Green LED for free slot
- Red LED for occupied slot
- Alert LED for long obstruction

## Components
- Arduino Uno
- HC-SR04 Sensor
- LEDs
- Resistors

## How it Works
The sensor measures distance. If an object is detected within 10cm, the slot is occupied.

## Output
- Green LED → Free
- Red LED → Occupied
- Alert LED → Long obstruction
